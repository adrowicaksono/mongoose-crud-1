## Books

List of Basic routes:

| Route                  | HTTP   | Description                 |
|------------------------|--------|-----------------------------|
| /book                  | GET    | Show list of books          |
| /book/add              | POST   | Create a book               |
| /book/edit/:isbn       | PUT    | Update a book with new info |
| /book/delete/:id       | DELETE | Delete a user               |
